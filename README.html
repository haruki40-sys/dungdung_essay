GitHub Pages 업로드 안내

ZIP 안 파일을 GitHub 저장소 최상단에 올리면 됩니다. 음악은 assets/audio 폴더에 넣으면 각 에세이 페이지와 연결됩니다.

음악 파일명

essay-1.mp3, essay-2.mp3, essay-3.mp3, essay-4.mp3 형식으로 넣어주세요. wav도 지원합니다.

1. GitHub에서 새 repository 생성

2. ZIP 안 파일을 전부 업로드

3. Settings → Pages → Deploy from a branch 선택

4. main / root 저장

5. 표시된 Pages 주소로 접속
